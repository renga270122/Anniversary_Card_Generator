## Common Questions

### Q: Do I need an API key to use the app?
**A:** No! The app works perfectly without it. AI features are optional.

### Q: Is my API key safe?
**A:** Yes! It's stored only in your browser, never on our servers.

### Q: Can I remove my API key?
**A:** Yes! Clear your browser's local storage or just delete the saved key in the app.

### Q: What if I forget my API key?
**A:** No problem - just get a new one from makersuite.google.com. You can have multiple keys.

### Q: How many messages can I generate?
**A:** Unlimited with templates! AI has a rate limit (60 per minute) - more than enough.

### Q: Can I use this offline?
**A:** Yes! Everything works offline except AI features.

### Q: What formats can I export?
**A:** PNG images (high-resolution) and text sharing.

## Troubleshooting

### Cards not generating?
- Select an event type first
- Enter recipient name
- Choose a tone
- Try again

### AI buttons not showing?
- Refresh the page
- Check your API key is pasted correctly
- Make sure you clicked "Save"

### API key invalid?
- Copy the entire key (no extra spaces)
- Get a new one from makersuite.google.com
- Make sure it starts with "AIzaSy"

### Image download not working?
- Try a different browser
- Check browser's download settings
- Allow popups if requested

### WhatsApp not opening?
- Have WhatsApp installed
- On mobile, it should open automatically
- On desktop, you'll get a prompt
